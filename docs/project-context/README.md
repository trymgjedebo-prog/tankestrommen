# Prosjektkontekst

Denne mappen er **prosjektminne og kontekst** for utviklere og agenter: stabil oversikt som ikke avhenger av chat-historikk.

## Tankestrømmen i korte trekk

Tankestrømmen analyserer inndata som **tekst, PDF og bilde** og produserer **strukturerte forslag** tilpasset Foreldre-appen.

## Viktige output-typer

- **Calendar proposals** — hendelser med dato/tid der det lar seg gjøre
- **Tasks / deadlines** — oppgaver og frister (f.eks. Spond-svar)
- **embeddedSchedule / delprogram** — flerdagersplan (cup, helg) som undersegmenter på parent-event
- **Highlights** — tidsforankrede eller programmessige høydepunkter per dag
- **Notes** — fri tekst og praktisk innhold som ikke skal bli feilaktige «programlinjer»
- **bringItems** — utstyr/pakkeliste der det er relevant
- **Tentative / conditional days** — dager som avhenger av resultat, sluttspill eller ukjent tidspunkt

Se også `architecture.md`, `testing-evals-and-ci.md` og de øvrige filene i denne mappen.
